LWF Loader

Loads LWF in a simple and elegant way.

Download .zip Download .tar.gz

LWF Loader provides a simple access to LWF features with minimum effort. It helps developers to quickly test and run their LWF project. LWF currently ONLY supports HTML5. LWF Loader is used along with LWF library.

How to use

Simply clone or download project to your environment

git clone git@github.com:gree/lwf-loader.git

Download underscore.js or any compatible library such as Lo-Dash.
Include these files to your project with LWF files and run:

// your settings
var setting = {
  lwf:'resource.lwf',
  prefix: "./",
  privateData: {}
}; 

var element = document.getElementById('lwf-element');
var lwfLoader = new window.LwfLoader();

lwfLoader.playLWF(element, setting);
